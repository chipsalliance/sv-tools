# sv-tools

A SystemVerilog toolchain / suite, encompassing a number of open source tools for working with SystemVerilog and UVM, meant to enable open source development of SystemVerilog/UVM codebases.

## Structure

The SV Tools project encompasses a number of tools, including:

* [SV tests](https://github.com/chipsalliance/sv-tests), a SystemVerilog and UVM testing suite and underlying infrastructure to track and improve SV/UVM support in various open source tools including linters, formatters, parsers, simulators etc.
* [Verible](https://github.com/chipsalliance/verible), a widely-used SV linting/formatting tool & LSP, with a VS code plugin 
* [Synlig](https://github.com/chipsalliance/synlig), a synthesis tool with SystemVerilog support
* [RISC-V DV](https://github.com/chipsalliance/riscv-dv), a SystemVerilog based DV testing framework for RISC-V cores
